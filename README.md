# bait.name — Official Website Repository

This repository contains the source code for the official website hosted at [bait.name](https://bait.name).

## Overview

This is a static, single-page website presenting professional Cyber Security consultancy services, with a specialisation in Palo Alto XSOAR (Extended Security Orchestration, Automation and Response). The site is informational only and requires no backend, database, or user authentication.

## Live Site

[https://bait.name](https://bait.name)

## Services Covered

- Integration Development
- Classifications & Mapping Development
- Pre-Processing Rules Development
- Incident & Object Development
- Playbook Development
- Dashboards & Reports Development

## Technology

- Pure HTML/CSS/JavaScript — no frameworks, no dependencies
- Hosted via GitHub Pages (free tier)
- Custom domain managed via Namecheap
- HTTPS enforced via GitHub's automatic SSL certificate

## Important Note

All XSOAR development services are delivered exclusively on the **client's own XSOAR development tenant**. No shared or third-party tenant is used at any point during engagements.
